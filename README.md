# F1 Made Simple

A beginner-friendly educational website about Formula 1. This project built for my COMP 2100 capstone project at the University of West Georgia.

## Description

F1 Made Simple breaks down the world of Formula 1 racing into easy-to-understand guides. Topics include the 2026 season calendar, race weekend format, rules and regulations, points system, a glossary of terms, and a quiz for people with no prior F1 knowledge. The site also includes a shop page with two products powered by Snipcart.

## Pages

- `index.html` — Homepage
- `about.html` — What is Formula 1 and its history
- `season.html` — 2026 season calendar and championship format
- `raceweekend.html` — Practice, qualifying, race, and sprint format
- `rules.html` — Flags, safety car, track limits, pit stop rules
- `points.html` — Points system, standings, and penalties
- `glossary.html` — Beginner-friendly F1 terminology
- `quiz.html` — 10-question knowledge quiz and contact form
- `products.html` — Shop page with two F1 products (Snipcart checkout)

## E-Commerce

The shop uses [Snipcart](https://snipcart.com) for cart and checkout functionality. Products are defined with `data-` attributes on HTML buttons and no custom JavaScript. The public API key is included in the page.

## Usage

Open `index.html` in any browser to start, or visit the live site via GitHub Pages.
